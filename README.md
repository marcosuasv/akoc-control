# Proyecto AKOC-Control

## Sistema de Pagos y Cobranza para Venta Inmobiliaria AKOC

Este proyecto es un sistema web diseñado para optimizar y automatizar la gestión de pagos, cobranza y generación de estados de cuenta para la Venta Inmobiliaria AKOC.

El objetivo principal es centralizar la información financiera, mejorar la eficiencia operativa y ofrecer una plataforma moderna tanto para los administradores como para los clientes.

### Módulos Principales

* **Gestión de Pagos:** Registro y seguimiento de pagos de clientes.
* **Gestión de Cobranza:** Administración de saldos pendientes y cartera.
* **Estados de Cuenta:** Generación y consulta de estados de cuenta.
* **Administración:** Panel para la gestión de propiedades, clientes y usuarios del sistema.

### Estado del Proyecto

Actualmente, el proyecto se encuentra en la fase de análisis y definición de la arquitectura.
